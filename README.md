1-O que é banco de dados
R-E um componente básico de um arquivo é um elemento com significado no mundo real ,que compõe um sistema de arquivo, coletar ,gerenciar, acessar, atualizar e analisar com formas rápida e eficiente.
2- defina SGBO .liste suas vantagens.
R-São programas de computador é um software que permite criar gerenciar e acessar bancos de dados .
Vantagem-organizar os dados de segurança das informações controle de acesso,redução de redundância, facilidade de consulta e atualização,e recuperação de dados.
3- o que é um DBA e o que ele é 
r- E o profissional respomsavel por gerenciar o banco de dados 
respomsabilidades-instalar e configurar o bancos de dados, garantir a segurança dos dados ,fazer backup e recuperação, monitorar o desempenho,contrata o acesso dos usuários.
